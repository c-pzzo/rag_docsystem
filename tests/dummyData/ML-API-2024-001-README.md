# [ML-API-2024-001] Customer Churn Predictor

## Overview
REST API service for predicting customer churn probability using machine learning. Implements multiple models including Random Forest and XGBoost, with A/B testing capabilities and model performance monitoring.

## Key Features
- Real-time predictions
- Model versioning
- Performance metrics
- Batch prediction support

## Technical Stack
- FastAPI
- scikit-learn
- MLflow
- Redis
